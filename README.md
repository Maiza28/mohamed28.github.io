# Mohamed Maiza - Academic Website

Ce dossier contient les fichiers de votre site web académique dans le style mixte (Ana Balibanu + Maxence Mayrand).

## Fichiers principaux

- **index.html** : Page d'accueil
- **Research.html** : Page de recherche
- **Cv.html** : Page CV

## Pour utiliser votre site

1. Placez tous les fichiers HTML dans le même dossier
2. Ajoutez votre photo de profil nommée `Maiza Mohamed.jpg` dans ce même dossier
3. Ouvrez `index.html` dans votre navigateur

## Structure du style

Le design suit ces principes :

- **Typographie** : Police serif Georgia pour un look académique classique
- **Couleurs** : 
  - Liens bleus (#0066cc)
  - Texte principal (#222)
  - Background gris très clair (#fafafa)
- **Layout** : Container centré (750px) avec ombre subtile
- **Navigation** : Horizontale avec effet de soulignement au survol
- **Responsive** : S'adapte automatiquement aux mobiles

## Personnalisation

Pour modifier le style, recherchez ces sections dans chaque fichier HTML :
- Les couleurs dans la section `:root` ou directement dans le CSS
- Les polices dans `font-family`
- Les dimensions dans `max-width`, `padding`, etc.

## Contact

Pour toute question sur le design, référez-vous aux commentaires dans le code CSS.
